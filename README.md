Open source noting android app with self host encrypted online storage.

## Features
- [ ] Dark / light themes
- [ ] Material 3 colorful themes
- [ ] Online self-host server
- [ ] Encrypted storing (local/online)

## Platforms Support
- [x] Android
- [ ] iOS (less probable)
- [ ] Web
- [ ] Linux / Windows
